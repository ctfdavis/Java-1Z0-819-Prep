Topic 3: Object-oriented Programming Concepts

Topic goals

  1. learn about objects, instances, classes & constructors
  2. learn about differences between primitive types and reference types
  3. understand modifiers
  4. learn about non-abstract classes & abstract classes
  5. understand polymorphism
  6. learn about interfaces & functional interfaces
  7. learn about enums
  8. learn about inner, nested & anonymous classes
  9. learn about garbage collection
