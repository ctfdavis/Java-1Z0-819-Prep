Topic 1: Java Data Types

Topic goals

  1. understand primitive types in Java
  2. understand wrapper classes in Java
  3. understand auto-boxing & auto-unboxing
  4. understand type promotion and casting
  5. learn about operators and their precedence
  6. learn about var (local variable type inference) & lambda parameters

Topic 2: Controlling Programme Flow

Topic goals

  1. understand for & enhanced for loops
  2. understand while & do/while loops
  3. understand if/else statements
  4. understand switch statements

Topic 3: Object-oriented Programming Concepts

Topic goals

  1. learn about objects, instances, classes & constructors
  2. learn about differences between primitive types and reference types
  3. understand modifiers
  4. learn about non-abstract classes & abstract classes
  5. understand polymorphism
  6. learn about interfaces & functional interfaces
  7. learn about enums
  8. learn about inner, nested & anonymous classes
  9. learn about garbage collection

Topic 4: Exception Handling

Topic goals

  1. learn about different exception types
  2. learn about throw & throws
  3. learn to handle exceptions with try/catch/finally blocks
  4. learn to create custom exceptions

Topic 5: Arrays & Collections

Topic goals

  1. learn about arrays
  2. learn about various Collections types (List, Set, Map & Deque)
  3. learn about Comparator & Comparable Interfaces

Topic 6: Streams & Lambda Expressions

Topic goals

  1. learn to use lambdas to implement functional interfaces
  2. learn to use Streams to filter, transform & process data

Topic 7: Module System

Topic goals

  1. understand the rationale & structure of a module
  2. learn about different types of module
  3. learn to create module-info and its keywords
  4. learn to display information about modules
  5. learn about built-in JDK modules
  6. learn about migration
  7. learn about services

Topic 8: I/O API

Topic goals

  1. learn to read from and write to files using I/O streams
  2. understand the uses & differences between various streams
  3. understand serialization & deserialization
  4. learn to use NIO.2 to create & use Path objects
  5. learn to operate on files & directories using NIO.2 Files

Topic 9: JDBC

Topic goals

  1. learn to connect to databases with JDBC
  2. learn to perform CRUD operations
  3. learn to perform other database operations

Topic 10: Localization

Topic goals

  1. learn about DateTime API
  2. learn to use Locale
  3. learn to use resource bundles

Topic 11: Annotations

Topic goals

  1. learn to declare annotations
  2. learn about built-in annotations
  3. learn to use common annotations

Topic 12: Security

Topic goals

  1. learn to prevent resource leaks
  2. learn to mitigate injection, inclusion, and input validation
  3. learn to secure resource access
